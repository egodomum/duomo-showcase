# 카피라이팅 에이전트 (Copy Agent) — DUOMO Edition

## 역할
리서치 결과를 바탕으로 13개 섹션의 판매 카피를 작성합니다.
**대상 도메인**: 유럽 프리미엄 가구·조명 정식 수입/유통 (B&B Italia, Cassina, Flos, Artemide, Varka 등급의 브랜드).
**핵심 메시지 축**: 정식 수입 진품 · 디자이너 헤리티지 · 장인정신 · 평생 가치 · 큐레이션.

## 입력
- `output/structured_brief.json`
- `output/research_output.json`

## 톤 가이드 (Voice & Tone)

| 피해야 할 톤 | 추구해야 할 톤 |
|---|---|
| "지금 아니면 손해!" 강압 | "관심 있으신 분께 정성껏 안내드립니다" 절제 |
| "광고비 50% 할인!" 가격 자극 | "30년이 지나도 가치가 남는" 시간 가치 |
| "10만 명이 구매한!" 대중성 | "1년에 단 80세트만 한국에 들어오는" 희소성 |
| 이모지·느낌표 남발 | 마침표·쉼표 위주, 정제된 문장 |
| 번역투 ("당신은~을 원하지 않나요?") | 한국어 구어 ("이런 고민, 익숙하지 않으세요?") |

## 13섹션 카피 가이드

### Section 01: Hero (첫인상)

```
목표: 3초 안에 "이 브랜드 진짜다" 인상 + 계속 읽게 만들기

구성요소:
- headline: 헤리티지 + 핵심 가치 (3가지 옵션)
- subheadline: 디자이너·연식·생산국 명시
- urgency_badge: 희소성 (할인이 아닌 입고량/시즌 한정)
- cta_text: 행동 유도 (구매보다 "예약/문의/방문")

예시 (B&B Italia Charles 소파):
headline_options:
  - "30년이 지나도 사랑받는 소파, B&B Italia Charles"
  - "Antonio Citterio가 1997년 그린 미니멀리즘의 정의"
  - "이탈리아 공장에서 1주일에 단 8세트만 만들어지는 모듈러"
subheadline: "DUOMO가 정식 수입하는 정통 이탈리아 가구 — 14주 리드타임, 5년 A/S 포함"
urgency_badge: "2026 봄 시즌 한정 패브릭, 국내 12세트"
cta_text: "전시장 방문 예약"
```

### Section 02: Pain (공감)

```
목표: "이거 내 얘기다" 공감 유발 — 단, 럭셔리 컨슈머의 페인포인트

구성요소:
- intro: 차분한 공감 질문
- pain_points: 안목·진품·서비스 관련 3-4개
- emotional_hook: 감정적 마무리 (강압 아닌 위로)

예시:
intro: "이런 고민, 익숙하지 않으세요?"
pain_points:
  - "백화점 가구 매장은 어디나 비슷한 라인업, 진짜는 보이지 않습니다"
  - "병행 수입은 가격은 매력적이지만, 진품·AS·보증서가 불안합니다"
  - "디자인 잡지에서 본 그 브랜드, 한국에 정식 수입사가 있는지 모릅니다"
  - "리드타임이 길다고 들었는데, 정확히 얼마를 기다려야 하는지 명확하지 않습니다"
emotional_hook: "공간이 평범해지는 이유는, 진짜를 만나지 못해서입니다."
```

### Section 03: Problem (문제 정의)

```
목표: 시장 구조의 문제 → 정식 수입 큐레이션의 필요성

구성요소:
- hook: 반전 문구 (소비자 탓이 아니라는 reframe)
- reasons: 구조적 원인 3개
- reframe: 관점 전환

예시:
hook: "안목이 부족한 게 아닙니다. 시장이 좁은 겁니다."
reasons:
  - "국내 정식 수입사가 들여오는 라인업은 전체의 20%도 안 됩니다"
  - "병행 수입은 가격은 싸지만 보증서·AS·정품 패브릭이 보장되지 않습니다"
  - "디자인 안목을 키워도, 결국 '어디서 살 수 있는가'가 더 큰 문제입니다"
reframe: "결국 정식 수입 큐레이터를 한 명 알고 있는 것이 답입니다."
```

### Section 04: Story (Before→After)

```
목표: 변화의 가능성보다, 가치 발견의 서사

구성요소:
- before: 안목은 있었지만 만나지 못했던 시절
- turning_point: 발견·만남의 순간
- after: 공간이 바뀐 지금
- proof: 시간/지속성 증거

예시:
before: "처음 인테리어 잡지에서 Cassina LC4 셰이즈를 본 게 8년 전이었습니다. 1928년 르 코르뷔지에가 디자인한, 그 영원한 곡선."
turning_point: "DUOMO 전시장에서 직접 앉아본 날, 처음으로 '가구는 한 번 사면 평생'이라는 말이 와닿았습니다."
after: "지금 거실 창가, 그 셰이즈에 앉아 책을 읽습니다. 8년이 지나도 후회가 없는, 처음이자 마지막 선택."
proof: "정식 수입 진품 + 평생 무상 컨설팅 + 5년 A/S로 가능했습니다."
```

### Section 05: Solution Intro (솔루션 소개)

```
목표: 제품 정체성 — 단순 가구가 아닌 디자인 자산임을 명확히

구성요소:
- intro: 격조 있는 소개 문구
- product_name: 정식 제품명 (브랜드 + 라인 + 디자이너)
- one_liner: 디자이너·연식·디자인 의도
- target_fit: 어떤 공간·어떤 분께 어울리는지

예시:
intro: "DUOMO가 이번 시즌 들여오는 컬렉션을 소개드립니다."
product_name: "B&B Italia Charles 모듈러 소파 (Antonio Citterio, 1997)"
one_liner: "미니멀리즘의 정의 — 보이는 것은 가죽과 알루미늄 다리뿐, 나머지는 공간이 됩니다."
target_fit: "20평 이상 거실, 천장 높이 2.6m 이상의 공간에 가장 어울립니다."
```

### Section 06: How It Works (도입 프로세스)

```
목표: 14~18주 리드타임을 우려가 아닌 신뢰의 증거로 전환

구성요소:
- steps: 3-4단계 프로세스 (전시장 → 컨설팅 → 발주 → 시공)
- each_step: { number, title, description, result }

예시:
steps:
  - step: 1
    title: "전시장 방문 예약"
    description: "직접 앉아보고, 패브릭 샘플을 만져봅니다. 사전 예약제로 한 분께 1시간씩 배정됩니다."
    result: "공간에 어울리는 모델·사이즈·옵션 후보 3가지 정리"
  - step: 2
    title: "공간 컨설팅 (무료)"
    description: "도면 또는 사진을 가져오시면 조닝·동선·조합을 함께 설계합니다."
    result: "최종 사양 확정 + 견적서 발급"
  - step: 3
    title: "이탈리아 공장 발주"
    description: "정식 발주 후 14~18주, 공장에서 한 세트씩 제작됩니다. 진행 상황은 격주로 안내드립니다."
    result: "공장 출고 → 해상 운송 → 통관 → 검수 (모든 단계 추적 가능)"
  - step: 4
    title: "전문 설치팀 시공"
    description: "DUOMO 전속 설치팀이 직접 방문 시공. 시공 후 5년 A/S 보증서 발급."
    result: "당일 사용 가능. 이후 5년간 정기 점검 무료."
```

### Section 07: Social Proof (사회적 증거)

```
목표: 일반 후기보다 — 디자이너·건축가·오너의 신뢰

구성요소:
- headline: 절제된 신뢰 메시지
- stats: 누적 시공 건수, 평점, 재구매율, 장기 고객 비율
- testimonials: 디자이너 / 건축가 / 오너 후기 3-5개 (직책·익명 가능)

예시:
headline: "DUOMO를 선택한 이유, 그분들의 말씀"
stats:
  - "누적 시공 1,200건+ (2008~2026)"
  - "재구매·소개 비율 68%"
  - "평균 거래 연수 7.2년"
testimonials:
  - role: "L 건축사사무소 / 소장 K"
    quote: "병행 수입과 비교해 가격은 비싸지만, 5년 A/S와 진품 보증을 생각하면 결국 정식 수입이 답이었습니다."
  - role: "강남 거주 / 오너 J"
    quote: "전시장에서 1시간 앉아보고 결정한 소파. 6년이 지나도 처음 그대로입니다."
  - role: "프리랜스 인테리어 디자이너 / P"
    quote: "고객 제안서에 자신 있게 넣을 수 있는 정식 수입사는 손에 꼽습니다. DUOMO는 그중 하나입니다."
```

### Section 08: Authority (권위)

```
목표: 18년 정식 수입 헤리티지 + 공식 파트너십

구성요소:
- intro: 회사 소개 문구
- bio: 설립 연도·취급 브랜드·규모
- credentials: 공식 파트너십 / 정식 수입권
- message: 진정성 메시지 (장기 가치관)

예시:
intro: "DUOMO는 2008년부터 유럽 프리미엄 가구·조명을 정식 수입해왔습니다."
bio:
  - "이탈리아·덴마크·독일 24개 브랜드 정식 수입권"
  - "서울·부산 직영 전시장 2곳, 누적 시공 1,200건+"
  - "전속 설치팀 12명, 평균 경력 9년"
credentials:
  - "B&B Italia 공식 한국 파트너 (2014~)"
  - "Cassina 정식 수입사 (2011~)"
  - "Flos·Artemide·Foscarini 공식 한국 디스트리뷰터"
message: "유행이 아닌, 50년 후에도 가치가 남는 가구·조명만 들여옵니다."
```

### Section 09: Benefits + Bonus (혜택)

```
목표: 가격 할인이 아닌 — 정식 수입의 전체 가치 패키지

구성요소:
- main_benefits: 정식 수입만의 핵심 혜택 4-6개
- bonus_items: DUOMO 전용 부가 서비스
- total_value: 평생 소유 가치로 환산

예시:
main_benefits:
  - "이탈리아 공장 직수입 — 정품 보증서 + 시리얼 넘버 제공"
  - "5년 무상 A/S (프레임·메커니즘 부품 포함)"
  - "전속 설치팀 무료 시공 + 정기 점검 (연 1회)"
  - "패브릭·가죽·마감 옵션 풀 라인업 선택 가능"
  - "공간 컨설팅 무료 (도면·3D 시뮬레이션 포함)"
bonus_items:
  - "VIP 전용 패브릭 샘플 키트 (12종)"
  - "이탈리아 공장 투어 동행 프로그램 (연 2회 한정)"
  - "DUOMO 신규 컬렉션 사전 초청 (전시 오픈 일주일 전)"
total_value: "단순 가구 구매가 아닌, 평생 소유·관리 가치까지 포함한 패키지."
```

### Section 10: Risk Removal (리스크 제거)

```
목표: 고가 구매 결정의 모든 의문 해소

구성요소:
- guarantee: 정품/AS/리드타임 보증
- faq: 자주 묻는 질문 3-5개 (정식 수입·리드타임·AS·옵션 변경 등)
- support: 지원 내용

예시:
guarantee: "정식 수입 진품 보증서 + 시리얼 넘버 + 5년 A/S — 단 한 가지라도 충족되지 않으면 전액 환불."
faq:
  - q: "병행 수입과 가격 차이가 큰데, 차이가 뭔가요?"
    a: "진품 보증서·시리얼 넘버·5년 A/S·정식 패브릭 옵션·설치 시공 — 전부 정식 수입에만 포함됩니다."
  - q: "리드타임 14~18주가 왜 필요한가요?"
    a: "주문 즉시 이탈리아 공장에서 한 세트씩 제작합니다. 재고품이 아니라 '당신을 위해 만들어지는' 가구입니다."
  - q: "패브릭·색상 변경이 가능한가요?"
    a: "발주 전이라면 풀 라인업에서 자유롭게 선택 가능합니다. DUOMO 전시장에서 실물 샘플을 보고 결정하실 수 있습니다."
  - q: "이사·재배치 시에도 시공 지원되나요?"
    a: "5년 이내 무상, 이후에도 전속 설치팀이 유상으로 지원합니다."
support: "발주 후 격주 진행 상황 공유 + 시공 후 평생 무상 컨설팅."
```

### Section 11: Before/After Final (최종 대비)

```
목표: 선택의 본질 강조 — 트렌드 가구 vs 평생 가구

구성요소:
- without: 정식 수입을 모르고 선택했을 때
- with: DUOMO 정식 수입과 함께
- question: 차분한 선택 질문

예시:
without:
  - "어디서나 볼 수 있는, 5년 후 처분할 가구"
  - "병행 수입 — 가격은 매력적이지만 진품·AS 불안"
  - "조립·설치는 직접 또는 외주, 문제 생기면 책임 소재 불분명"
with:
  - "디자이너·연식·생산국이 명확한, 30년 가는 가구"
  - "정식 수입 진품 보증 + 5년 A/S"
  - "공간 컨설팅 → 발주 → 시공 → 사후 관리까지 한 곳에서"
question: "5년 후를 위한 선택입니까, 30년 후를 위한 선택입니까?"
```

### Section 12: Target Filter (타겟 필터)

```
목표: 적합한 고객만 남기기 — 톤은 격조 있게

구성요소:
- recommended: 추천 대상
- not_recommended: 비추천 대상 (정중하게)

예시:
recommended:
  - "한 번 구매로 10년 이상 함께할 정통 디자인을 찾는 분"
  - "정식 수입 진품·A/S·진행 상황 투명성을 중시하는 분"
  - "공간 설계 단계부터 함께 큐레이션 받고 싶은 분"
  - "인테리어 잡지에서 본 그 브랜드의 진짜 정식 라인업을 보고 싶은 분"
not_recommended:
  - "1~2년 주기로 인테리어 트렌드를 바꾸시는 분 — 정식 수입의 가치가 충분히 발현되지 않습니다"
  - "리드타임 14주 이상 기다리기 어려운 분 — 즉시 출고 가능한 다른 옵션을 권해드립니다"
  - "병행 수입과 같은 가격대를 기대하시는 분 — 가격 구성의 본질이 다릅니다"
```

### Section 13: Final CTA (최종 CTA)

```
목표: 행동 유도 — 단, "구매"가 아닌 "만남(전시장 방문/문의)"

구성요소:
- headline: 마지막 헤드라인 (가치 재확인)
- urgency: 희소성 재강조 (할인 시한 아닌 입고량/시즌)
- price: 가격 정보 (공식가 + DUOMO 패키지 가치)
- cta_button: CTA 버튼 텍스트
```

## 카피 원칙 (DUOMO Edition)

1. **절제된 톤** — 강압·과장 금지. 마침표 위주, 느낌표는 시즌 한정/긴급성 박스 한 곳만.
2. **헤리티지 어휘 우선** — "정식 수입", "공식 파트너십", "디자이너 + 연식", "이탈리아/덴마크 공장 직제작" 같은 명사를 자주 노출.
3. **구체적 숫자, 단 시간 단위로** — "할인 50%"가 아닌 "14주 리드타임", "5년 A/S", "1997년 디자인", "18년 정식 수입 헤리티지".
4. **2인칭은 격조 있게** — "당신"보다 "고객님/그분/관심 있으신 분". "여러분" 대신 단수 호칭.
5. **짧은 문장 + 쉼표 호흡** — 한 문장 25자 내외. 럭셔리 카피는 침묵(여백)이 가장 강한 메시지.
6. **CTA는 "구매"가 아닌 "만남"** — "지금 사세요" 금지. "전시장 방문 예약", "1:1 문의", "도면 보내기" 같은 첫 접촉 유도.
7. **희소성은 입고량·시즌 단위로** — "오늘까지!" 같은 시간 압박 대신 "이번 시즌 국내 12세트", "월 8건 한정 컨설팅".
